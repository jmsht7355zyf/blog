      sass基础入门

      # 安装

      npm install  -g sass -D

      # sassScript

      # 变量

      定义变量：

      $boxWidth: 1200px;
$boxColor: white;

      调用变量：

      width:$boxWidth
color:$boxColor

      # 变量作用域

      变量支持块级作用域，嵌套规则内定义的变量智能在规则内使用（局部变量），不在嵌套规则内定义的变量则可以在任意处使用（全局变量），将局部变量转换为全局变量可以添加!global声明。

      # 其他语法

      # 插值语法

      通过#{}可在选择器、属性名和属性值中使用变量。

      大多数情况下，使用属性值可能不如直接使用变量方便，但使用#{}可避免sass运行运算表达式，直接编译

      $name:foo;
$attr: border;
p.#{name}{
    #{attr}-color:$name;
}

//编译结果
p.foo{
    border-color:foo;
}

      # & in sassScript

      &为父选择器

      button {
	background-color: red;
	&hover {
	background-color: green;
	}
}

      # 继承@extend

      .a{
    color:red;
}
.b{
    @extend .a
}

      # 三元运算符

      p{
	color: if(1+1 == 2,green,yellow);
}

      # @if

      p{
    @if 1-1 != 2 {
        color: red;
    }@else{
        color:blue;
    }
}

      # @for

      @for $i from 1 through 3 {
    .item-#{$i}{width:2em * $i;}
}

//编译结果
.item-1{
    width:2en;
}
.item-2{
    width:2en;
}
.item-3{
    width:2en;
}

      # @while

      $i:6;
@while $i > 0 {
    .item-#{$i}{width:2em * $i;}
    $i: $i - 2;
}
//编译结果
.item-6{
    width:2en;
}
.item-4{
    width:2en;
}
.item-2{
    width:2en;
}

      # @mixin

      //定义
@mixin box {
    color: red;
    font-size:16px;
}
//调用
box {
    @include box;
}

//添加参数
@mixin box （$text-color,$text-size）{
    color:$text-color;
    font-size:$text-size;
}
box {
    @include box(red,16px);
}

      # 自定义函数@function

      $colors:(light:white,dark:black);
@function color($key){
    @return map-get($colors,$key);
}
body{
    background-color: color(light);
}
